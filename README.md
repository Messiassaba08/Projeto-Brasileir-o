## Introdução
Esta documentação tem como objetivo principal explicar as decisões voltadas a analise de dados, do uso da estatística e da ferramenta de manipulação de dados. Sendo assim, não irei especificar sobre a criação do dashboard pelo Figma, mas irei deixar o link dos vídeos que usei para auxiliar no uso da ferramenta logo acima. Também deixei o link do curso gratuito de Power BI que realizei para criação do projeto. Lembrando que apenas a vizualização dos vídeos não irá desenvolver suas habilidades com as ferramentas se não houver prática. Esse é o objetivo da criação desse projeto. :)
